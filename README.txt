<3 by Hannah Brjesson and Per Jonsson
A lovable imperative interpreted programming language

To run source code files written in <3 you need at least Ruby 1.9.3

Please name your source files with the extension .three

To run programs written in <3 write the following command in a terminal:
ruby 3.rb
... and the interpreter will start. You will then be asked to specify a source code file to run.

There is an example file provided called 3.three

Please refer to Three-Manual.pdf for further instructions
